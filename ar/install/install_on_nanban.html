charites on unvt/nanban

Charites runs on Docker with one of UNVT tools named unvt/nanban. Charites is already installed in the tool, so you can use it by just staring unvt/nanban.

$ docker run -ti --rm -v ${PWD}:/data -p 8080:8080 unvt/nanban
$ cd /data
